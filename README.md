# ShoppingList

Android-приложение для мониторинга своих покупок.

## Функционал

Приложение содержит 4 основных экрана

1. Настройки приложения.
2. Корзина.
3. Чеки пользователя.
4. Заметки пользователя.

## Инструменты реализации

- Android Studio
- Kotlin
- Coroutines
- Flow
- Jetpack Compose
- Hilt
- Room
- MVVM
- Datastore
- GSON
- Constraint Layout
- Navigation Compose
